Configure Github Pages

Publish date: Apr 22, 2019
Tags: github pages configuration

To setup github-pages do the following:

  • on the settings page scroll down to the github pages section
  • under source select “master branch/docs folder”
  • no need to select a theme or a custom theme

Once everything is setup the page can be published doing make publish. This will commit everything in the docs folder with the message published.

Notes:

  • currently github pages does not support publishing git lfs files.
    The simplest workaround is to not use git lfs for referenced files. Should you accidentally have it turned on you can remove the files, use git lfs untrack ~bad files~ and readd them.

  • I chose a single message for publish so it would be easy to filter out.
